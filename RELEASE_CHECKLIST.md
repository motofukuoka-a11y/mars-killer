## リリース前に実環境で確認する項目

1. GitHub Pagesをオンラインで開き、Service Workerの登録成功を確認する。
2. 一度全画面を開いた後、DevToolsをOfflineにして再読込みする。
3. iPhone Safariで発着駅候補、旅客人数、経由駅、区間設定を操作する。
4. ライト／ダーク両モードで文字とエラー表示を確認する。
5. Console Errorおよび未処理Promise rejectionが0件であることを確認する。
6. 実データを使い、運賃・料金・割引・営業規則を教科書と照合する。

未収録の料金表・割引規則は推測計算せず、警告または計算不能として扱う。
